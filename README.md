# ProjectErectus
Sensor Data From Android Smartwatches


Arff file format

@relation (Dataset Name)

@attribute DeviceID string
@attribute TimeStamp numeric
@attribute x numeric
@attribute y numeric
@attribute z numeric
@attribute Accuracy numeric
@attribute DataSource string
@attribute DataType numeric

@data
1,2,3,4,5,6,7 following csv format.

DeviceID 
•	Basic device ID of the Nexus 5 phone used for experiment. 
TimeStamp 
•	Linux kernel Timestamp
X 
•	Accelerometer X axis value
Y
•	Accelerometer Y axis value
Z
•	Accelerometer Z axis value
Accuracy 
•	Degree of accuracy (0 meaning high accuracy)
Data Source
•	Source of sensor
 Data Type
•	Packet type
